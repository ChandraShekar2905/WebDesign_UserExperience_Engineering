# League of Legends Champions Website

## Features
1.HTML Elements Used:
   - Table: Displays champion stats.
   - Form: Includes text, email, password fields, and a datalist.
   - Images: Used alongside `<figure>` and `<figcaption>`.
   - Video: Embedded media for a better user experience.
   - Hyperlinks: Includes `mailto` and `tel` links.
   - Details & Summary: Provides collapsible content.

2. Styling:
   - External CSS ensures a consistent and visually appealing layout.

3. Responsive Design:
   - Breaks the page into sections for better readability.

4.Tags Used: <html>, <head>, <title>, <body>, <header>, <section>, <table>, <form>, <datalist>, <figure>, <audio>, <video>, <aside>, <details>, <summary>, <a>, <footer>.

5.This project is a League of Legends Champions Showcase Website that provides information about popular champions from the game. It features sections for champion stats, media content, and contact information.
The website is designed with a clean layout and includes navigation links and multimedia elements. 
It uses HTML5 and an external CSS file to ensure a structured and visually appealing user experience.
