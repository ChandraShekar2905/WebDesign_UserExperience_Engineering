# Portfolio Website

## Tags Used
- header, section, footer**: Used for semantic structure.
- nav, table, form**: Added navigation, tables, and forms for user interaction.
- img: For displaying the image gallery.

## Features
- Favicon: Added a favicon for branding.
- Testimonials Section: Interactive testimonials with hover effects.
- Responsive Image Gallery: Flexbox-based image gallery that adjusts for different screen sizes.
- CSS Selectors: Styled the table using multiple CSS selectors.
- Responsive Design: Implemented media queries for iPad (768px) and iPhone (375px) responsiveness.
